# test-entelgy-openbank

This is a test to join as Front-end in OpenBank.

### Technologies:

- Front: React with Hooks, SCSS, RTL and jest (testing).
- Others: Lint and Prettier

### To run the project:

Into test-entelgy-openbank folder:

- run: `npm start`
- for tests run `npm test`
